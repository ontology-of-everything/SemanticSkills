# 查询说明（模拟，未调用 live hcloud）

这是**资源包抵扣与按需账单差异**类问题：需要把「仍在出的按需账单」和资源包是否在同样维度、同一时间窗抵扣对齐。

## 结果表（示例占位）

| 步骤 | 证据类型 | 建议只读入口（KooCLI BSS） |
| --- | --- | --- |
| 1 | 资源包实例列表 | List 风格资源包/API 对应命令（占位） |
| 2 | 资源包余量 | Show/List 余量 |
| 3 | 抵扣/用量明细 | 抵扣记录或用量抵扣流水 |
| 4 | 同账期按需资源详单 | 资源级账单/明细 |

小结：你已购买资源包后仍看到按需计费，常见于资源包适用范围、区域、规格、用量类型不匹配，或抵扣存在出账延迟；在未拉取上述四类只读证据前，不能归因到单一结论。

### 用户可读说明

时间：默认核对**当前聚焦账期**（若你只说了「这个月」，按当前登录账号、当前控制台所选账期理解）。账号：当前认证 profile。上述表为占位行；真实应答需在脱敏后以表格列出账期、产品、region、计费项、抵扣前后金额等。

## 已证实事实 / 可能解释 / 待核验边界

- **已证实事实（当前会话）**：你已表述「已购资源包且仍有按需扣费」，属客户侧陈述；账务侧数值未核验。
- **可能解释**：产品不在包内 SKU；region/企业项目不符；规格或用量类型（如流出 vs 流入）不匹配；资源包已过生效/失效时间；抵扣记录在详单中下账晚于概览截图时间。
- **待核验边界**：具体包 SKU 与按需行的 `product`、`region`、`resource_id`（输出需脱敏）是否逐项对齐；所选账期与导出 CSV 切段是否一致。

**安全边界**：不进行支付、退订、退款或删除资源操作；不代为承诺退费，仅整理只读证据后建议通过工单/账号经理走商务或账务复核路径。
